# 반응형 디자인을 구현하면서 알게 된 개념, 사용한 CSS 속성
## 반응형 디자인을 구현하며 알게 된 개념
반응형 디자인이란 단순히 요소 크기만 줄이는 것이 아니라 레이아웃 구조 자체를 화면 크기에 맞게 바꾸는 것  

화면이 작아질 수록 우선순위에 따라 높은 것부터 보여주는 식으로 설계해야 함

## 사용한 CSS 속성
### @media
화면 크기에 따라 다른 스타일 적용

### flex
네브바 내부 요소 가로 정렬

### justify-content
flex 내부 요소의 가로 정렬 방식

### align-items
flex 내부 요소의 세로 정렬 방식

### gap
flex 내부 요소 사이의 간격

### display: none
화면이 작아졌을 때 메뉴를 숨기기 위해

### width
요소의 너비를 설정할 때

### padding
요소 안 쪽의 너비를 줄 때

### border / border-radius
네브바 버튼의 테두리, 둥근 모양

### box-shadow
그림자를 주기 위해

### background-color, background-image, background-size
배경색과 배경 패턴을 만들 때

### font-size, font-weight, color
텍스트의 크기, 굵기, 색상을 조절할 때

### white-space: nowrap
텍스트가 줄바꿈되지 않도록 하기 위해

### flex-shrink: 0
화면이 줄어들 때 요소 모양을 유지하기 위해
