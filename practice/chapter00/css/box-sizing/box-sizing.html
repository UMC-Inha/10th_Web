border vs outline

border일 때

width: 100px; height: 100px; border: 10px로 지정

- border-box

border는 지정한 width, height 내부 / 박스 내부에 추가됨

=> 총 100px / content 영역 80px

- content-box

border는 지정한 width, height 외부 / 박스 내부에 추가됨

총 120px / content 영역 100px

outline일 때

width: 100px; height: 100px; outline: 10px로 지정

- border-box

- content-box

테두리가 지정한 width, height 외부 / 박스 외부에 추가됨 => 총 100px (20px은 박스 영역으로 잡히지 않음)

outline은 box model 계산에 포함되지 않으므로 border-box든 content-box든 width / height는 그대로이고 outline은 박스 외부에 그려짐
