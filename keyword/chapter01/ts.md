- 함수 선언식의 특징에 대해 정리해주세요!
    
    function 키원드를 사용하여 이름을 붙여 선언합니다.
    
    함수가 선언된 줄보다 위쪽에서 함수를 호출해도 에러 없이 실행됩니다.
    
    함수가 어떻게 호출되느냐에 따라 this가 가르키는 대상이 동적으로 변합니다.
    
- 화살표 함수의 특징에 대해 정리해주세요!
    
    변수에 할당하는 방식이므로. 함수를 만들기 전에 호출하면 에러가 발생합니다. 
    
    자신만의 this를 가지지 않고 자신을 감싸고 있는 외부의 this를 가져다가 쓴다요 하네요?


- 타입 스크립트에만 존재하는 타입 🍠
    - any 🍠
        
        어떤 타입이든 다 OK
        
    - unknown 🍠
        
        any 처럼 어떤 타입이든 값을 넣을 수 있지만 타입을 확인하기 전까지는 다른 곳에 할당하거나 사용할 수가 없습니다.
        
    - void 🍠
        
        반환 값이 없는 타입입니다. 보통 함수의 반환 타입으로 씁니다.
        
    - never 🍠
        
        값을 갖을 수 없거나, 무한 루프를 돌거나, 항상 에러를 던지는 함수의 반환 타입으로 씁니다? 혹은 논리적으로 발생할 수 없는 타입을 정의할때 사용합니다.
