# 기타 태그 추가 정리해보기 🍠
**폼 관련 태그들:**

- `<form>` : 사용자 입력 양식
- `<input>` : 다양한 입력 필드
- `<button>` : 클릭 가능한 버튼
- `<select>`, `<option>` : 드롭다운 선택 메뉴

**텍스트 관련 태그들:**

- `<strong>` : 중요한 텍스트 (굵게)
- `<em>` : 강조 텍스트 (기울임)
- `<mark>` : 하이라이트된 텍스트

**추가로 태그를 정리해주세요:**

**텍스트**

- `<h1>` : 가장 큰 제목, 페이지에서 제일 중요한 제목에 사용
- `<h2>` : 중간 제목, `<h1>`보다 한 단계 아래 제목
- `<h3>` : 세부 제목, `<h2>` 보다 한 단계 아래 제목 (`<h6>` 까지 존재)
- `<p>` : 문단 (paragraphy), 일반적인 본문 글을 쓸 때 사용
- `<span>` : 큰 의미는 없고 글자 일부만 묶고 싶을 때 사용
    - 줄 바꿈이 안 되기 때문에 텍스트 일부만 꾸밀 때 주로 사용

**목록**

- `<ul>` : 순서 없는 목록 (앞에 - 가 붙음)
- `<ol>` : 순서 있는 목록 (앞에 번호가 붙음)
- `<li>` : `<ul>`, `<ol>` 안에서 쓰는 목록 각각의 항목

**표**

- `<table>` : 표 전체를 감싸는 태그
- `<thead>` : 표의 머리 부분 (제목)
- `<tbody>` : 표의 본문 부분 (실제 데이터)
- `<tfoot>` : 표의 마무리 부분 (합계, 요약 등)
- `<th>` : 표의 제목, 굵게 표시
- `<tr>` : 표의 한 줄
- `<td>` : 표의 일반적인 데이터 칸

```html
// 표 관련 태그들의 예시
<table>
	<thead>
		<tr>
			<th>이름</th>
			<th>나이</th>
		</tr>
	</thead>
	<tbody>
		<tr>
			<td>서현</td>
			<td>25</td>
		</tr>
		<tr>
			<td>엘리</td>
			<td>25</td>
		</tr>
	</tbody>
	<tfoot>
		<tr>
			<td>총</td>
			<td>2명</td>
		</tr>
	</tfoot>
</table>
```

**입력**

- `<textarea>` : 여러 줄 입력 칸
    - `<input>`은 한 줄만 입력되기에 긴 글을 적을 때 사용

# Sementic Tag를 잘 활용하였을 때 장점은? 🍠
- 개발자가 개발할 때 태그 이름만으로 내용/위치를 이해하기 쉬워진다. 코드 가독성이 좋아지고, 유지보수가 쉬워진다.
    
- 검색엔진 최적화(SEO)에 유리하다. 검색엔진은 HTML 구조를 보고 페이지 내용을 분석하는데, 시멘틱 태그를 쓰면 검색엔진이 이 문서의 중요한 부분이 어디인지 더 잘 파악할 수 있다. 좋은 구조를 만들면 검색엔진에 유리해져 검색 순위가 오를 수 있다.
    
- 웹 접근성에 도움이 된다. 스크린리더 같은 보조기기가 페이지를 읽을 때 시멘틱 태그를 기준으로 읽기 때문에 시각장애인 같은 보조기기를 이용하는 사용자도 페이지를 이해하기 쉬워진다.
