- 기타 태그 추가 정리해보기 🍠
    
    **폼 관련 태그들:**
    
    - `<form>` : 사용자 입력 양식
    - `<input>` : 다양한 입력 필드
    - `<button>` : 클릭 가능한 버튼
    - `<select>`, `<option>` : 드롭다운 선택 메뉴
    
    **텍스트 관련 태그들:**
    
    - `<strong>` : 중요한 텍스트 (굵게)
    - `<em>` : 강조 텍스트 (기울임)
    - `<mark>` : 하이라이트된 텍스트
    
    **추가로 태그를 정리해주세요:**
    
    - `<ol>` : 순서가 있는 목록
    - `<ul>` : 순서가 없는 목록
    - `<li>` : 목록내의 항목
    - `<span>`  : 인라인 스타일을 위한 텍스트 그룹화
    - `<br>` : 줄 바꿈

    - **`Sementic Tag`**를 잘 활용하였을 때 장점은? 🍠
    
    시메틱 태그를 사용하면 그 태그가 담고 있는 의미를 빠르고 구체적으로 전달할 수 있는 장점이 있습니다.

    - 그러면, **`inline-block`**은 어떠한 방식으로 동작할까요? 🍠
    
    인라인 성질과 블록의 성질을 반반씩 섞어놓은 겁니다. 인라인 처럼 옆 요소들과 나란히 배치됩니다. 그러면서도 블록처럼 width나 heigh,margin을 마음대로 조절할 수 있습니다**.**
