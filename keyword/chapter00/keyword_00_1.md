기타 태그 추가 정리해보기 🍠
**시간, 정보 관련 태그들:**

- `<time>` : 날짜나 시간 표시
- `<address>` : 연락처 정보

**삽입 관련 태그들:**

- `<object>` : 외부 문서, 매체, 플러그인 등을 웹페이지에 삽입
- `<iframe>` : 외부 문서를 해당 영역에 삽입

**용도가 바뀐 태그들:**

- `<b>` : 굵은 글씨 표현 → `<strong>`보다 약한 강조 의미
- `<s>` : 취소선
- `<u>` : 밑줄
- `<i>` : 이탤릭체
- `<hr>` : 가로줄

Sementic Tag를 잘 활용하였을 때 장점은? 🍠

1. **가독성 향상** -태그 자체가 의미를 포함하고 있어 코드를 읽을 때 이해가 수월해짐
2. **검색엔진 최적** -검색엔진이 페이지 구조를 더 정확하게 이해할 수 있음
   ⇒ 의미가 명확한 태그를 사용하므로 검색 결과 노출에도 도움이 될 수 있음
3. **접근성 향상** -보조 기술이 페이지 구조를 이해하기 쉬워짐
   ⇒ 스크린 리더와 같은 시각장애인을 위한 도구 사용이 쉬워져서 시각 장애인들의 사이트 이용이 수월해짐
4. **유지보수가 좋아짐**
   =>의미있는 코드 사용 시 코드의 구조 한눈에 쉽게 파악과 이해 가능
   =>협업할 때 어떤 부분이 어떤 역할 수행하고 있는지 빠르게 이해가능하므로 협업 효율과 코드의 유지 보수성 크게 향상시킴
