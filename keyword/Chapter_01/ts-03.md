# 타입 스크립트에만 존재하는 타입 🍠 (any 🍠, unknown 🍠, void 🍠, never 🍠)

## any
아무 타입이나 다 허용하는 타입
### 특징
- 어떤 값이든 다 넣을 수 있음
- 타입 검사도 안 함
### 장점
- 임시로 빠르게 코드 작성하기 편함
### 단점
- 타입스크립트를 쓰는 의미가 줄어듦
- 에러를 미리 못 잡을 수 있음

## unknown
아무 값이나 다 받을 수 있는 타입이지만 any와 다르게 바로 사용할 수 없고 확인 후에 써야 함
### 특징
- 어떤 값이든 할당 가능
- 바로 메서드 호출을 하거나 연산은 불가능
- 타입 확인 후 사용해야 함
### any와의 차이점
- any
    - 아무거나 넣고 바로 써도 됨
    - 아무거나 넣을 수는 있지만 쓸 때 검사 필요

## void
아무것도 반환하지 않는 함수의 반환 타입
### 특징
- 함수가 값을 return하지 않을 때 사용
- 출력만 하거나 내부 처리만 하는 함수에 사용

## never
절대 값을 반환할 수 없는 경우에 쓰는 타입
### 특징
- 함수가 끝까지 정상 종료되지 않을 때 사용
- 무한 반복 함수에 사용
### void와의 차이점
- void
    - 함수 실행은 끝남
    - 반환값은 없음
- never
    - 함수가 정상적으로 끝나지 않음
    - 반환 자체가 불가능함
