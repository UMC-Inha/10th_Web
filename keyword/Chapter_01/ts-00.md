# null과 undefined의 차이점에 대해 직접 작성해주세요 🍠
둘 다 비어 있거나 없음을 나타내는 값
## undefined
값이 아직 정해지지 않았을 때  
변수를 선언만 하고 값을 넣지 않았거나, 객체에 존재하지 않는 속성에 접근했을 때 나타남
```
let name;
console.log(name); // undefined
```

## null
값이 없음을 넣었을 때  
비어있음을 명시적으로 표현할 때 사용
```
let user: string | null = null;
console.log(user); // null
```
